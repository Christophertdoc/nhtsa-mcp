# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

This is an MCP (Model Context Protocol) server for the NHTSA (National Highway Traffic Safety Administration) API. The project is in its initial stages — no source code, build system, or test infrastructure exists yet.

## IDE Configuration

The `.vscode/settings.json` is configured with `claudeCode.useTerminal: true`, meaning Claude Code uses the integrated terminal rather than the editor for commands.
