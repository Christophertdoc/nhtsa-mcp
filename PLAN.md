# NHTSA MCP Server — Implementation Plan

> **Task tracking:** See [TASKS.md](./TASKS.md) for the full task list with per-task details, milestone groupings, and status tracking.

## Context
Build a Python + FastAPI MCP server that provides structured, safe, rate-limited access to five NHTSA API surfaces (vPIC VIN decoding, NCAP Safety Ratings, Recalls, Complaints, Car Seat Inspection Stations). The server exposes 10 MCP tools with strict Pydantic v2 input validation, in-memory TTL caching, per-IP rate limiting, and both HTTP/SSE and stdio MCP transports. A Typer-based CLI test harness and an LLM tool-calling agent are included. The project currently has no source code.

## Input Validation (`app/models/inputs.py`)

All models: `ConfigDict(str_strip_whitespace=True, frozen=True)`, Pydantic v2 `AfterValidator` chain.

| Field | Constraint |
|---|---|
| `vin` | exactly 17 chars, `.upper()`, no I/O/Q, alphanumeric only |
| `model_year` | `int`, `ge=1980`, `le=current_year+1` |
| `make` / `model` | `min=1, max=64`, `[A-Za-z0-9 \-]` only, title-case normalize |
| `vehicle_id` | `int > 0` |
| `campaign_number` | `min=5, max=20`, pattern `r"^\d{2}[A-Z]\d{6}$"` |
| `odi_number` | digits only, `min=5, max=12` |
| `zip` | normalize ZIP+4 → 5 digits; assert 5-digit result |
| `state` | `.upper()`, must be in 56-code allowlist (50 states + DC + territories) |
| `lat` | `ge=-90.0, le=90.0` |
| `long` | `ge=-180.0, le=180.0` |
| `miles` | `ge=1, le=200`, default 25 |
| `lang` | allowlist: `en`/`english` → `"english"`, `es`/`spanish` → `"spanish"` |

```python
@dataclass
class AppContext:
    vpic_client: VPICClient
    nhtsa_client: APIClientNHTSA
    caches: dict[str, AsyncTTLCache]
    rate_limiter: RateLimiter
    settings: Settings

@asynccontextmanager
async def app_lifespan(server: FastMCP) -> AsyncIterator[AppContext]:
    # Creates two shared httpx.AsyncClient instances (one per base URL)
    # Creates one asyncio.Semaphore for concurrency cap
    # Creates per-domain AsyncTTLCache instances
    # Yields AppContext; both clients closed on shutdown
    ...

mcp = FastMCP(name="NHTSA MCP", lifespan=app_lifespan)
# Tool modules imported here (import triggers @mcp.tool() registration)
app = FastAPI()
app.mount("/mcp", mcp.sse_app())  # SSE transport
# GET /health -> {"status": "ok"}
```

**Key design choices:**
- `FastMCP` from `mcp[cli]` (official SDK) supports both stdio and HTTP/SSE from same codebase
- Two separate `httpx.AsyncClient` instances (one per upstream base URL), created at lifespan and shared across all requests — no per-request connection overhead
- `asyncio.Semaphore(max_concurrent_upstream_requests)` wraps every `_get()` call in `base_client.py`
- All upstream path construction uses hardcoded base + validated path fragments — no user input in base URL (SSRF prevention)

## Client Layer (`app/nhtsa_clients/`)

### `base_client.py`
- `_assert_allowlisted_path(path)`: checks path starts with one of `ALLOWED_PATH_PREFIXES`; raises immediately if not
- `_get(path, params)`: acquires semaphore → retries via `tenacity` (`stop_after_attempt(3)`, `wait_exponential(min=1s, max=10s, multiplier=2)`) → maps status codes to typed exceptions
- Retry triggers on: `httpx.TimeoutException`, `httpx.ConnectError`, `UpstreamServerError` (5xx)

### `vpic_client.py`
```
ALLOWED_PATH_PREFIXES = ["/vehicles/DecodeVinValues/", "/vehicles/DecodeVinValuesExtended/"]
decode_vin(vin, model_year, extended) → GET /vehicles/Decode[Extended/]{vin}?format=json[&modelyear=YYYY]
```

### `api_nhtsa_client.py`
```
ALLOWED_PATH_PREFIXES = ["/SafetyRatings", "/recalls/recallsByVehicle", "/recalls/campaignNumber",
                          "/complaints/complaintsByVehicle", "/complaints/odinumber", "/CSSIStation"]
```
CSSI path building: `/CSSIStation/zip/{zip}[/lang/spanish][/cpsweek]`

## Security & Rate Limiting (`app/security/`)

### Rate Limiter (`rate_limiter.py`)
- Pure in-memory, `collections.deque` sliding windows per IP
- Key: `sha256(ip_address)` (never logs raw IP)
- Background `asyncio.Task` prunes stale buckets every 5 minutes

| Limit | Default | Env Var |
|---|---|---|
| Global per-IP per minute | 60 | `RATE_LIMIT_GLOBAL_PER_MINUTE` |
| VIN per-IP per minute | 10 | `RATE_LIMIT_VIN_PER_MINUTE` |
| Daily quota per-IP | 1000 | `RATE_LIMIT_DAILY_QUOTA` |

Rate limiter invoked **before** Pydantic validation in each tool (prevents validation cost from being exploitable).

### Sanitizer (`sanitizer.py`)
- Strips any key in `REDACTED_FIELDS = {"traceback", "stack_trace", "exception", "internal_url", "x-api-key", "authorization", ...}`
- Truncates any single string field > 50,000 chars
- `sanitize_error(exc)` maps exception type → safe user message; never exposes `__traceback__`

### Error Mapping
| Exception | HTTP Status | Code |
|---|---|---|
| `ValidationError` | 400 | `VALIDATION_ERROR` |
| `RateLimitExceededError` | 429 | `RATE_LIMIT_EXCEEDED` (+ `Retry-After`) |
| `UpstreamRateLimitError` | 429 | `UPSTREAM_RATE_LIMITED` |
| `UpstreamServerError` | 502 | `UPSTREAM_ERROR` |
| `UpstreamTimeoutError` | 504 | `UPSTREAM_TIMEOUT` |
| `UpstreamConnectError` | 502 | `UPSTREAM_UNREACHABLE` |
| `UpstreamClientError(404)` | 404 | `NOT_FOUND` |
| Unexpected | 500 | `INTERNAL_ERROR` (generic message only) |

## Caching (`app/security/cache.py`)

`AsyncTTLCache`: `asyncio.Lock` + `cachetools.TTLCache`. `get_or_fetch(key, fetch_fn)` pattern.

Cache keys derived from **normalized** inputs so `"toyota"` and `"Toyota"` share the same entry.

| Cache | TTL | Max Entries |
|---|---|---|
| VIN decode | 86,400s (24h) | 500 |
| Safety ratings | 3,600s (1h) | 200 |
| Recalls | 1,800s (30m) | 200 |
| Complaints | 900s (15m) | 200 |
| CSSI stations | 3,600s (1h) | 100 |

## LLM Agent (`cli/llm_agent.py`)

### Provider abstraction
```python
class LLMProvider(ABC):
    def complete_with_tools(messages, tools, system) -> tuple[str | None, list[ToolCall]]: ...

class AnthropicProvider(LLMProvider): ...
class OpenAIProvider(LLMProvider): ...

def get_provider(settings: Settings) -> LLMProvider: ...
```

### `run_agent` guardrails
1. **Tool allowlist**: only tool names returned by `/mcp/tools/list` are forwarded
2. **Schema validation**: tool arguments pass the same Pydantic validators locally before sending to MCP server
3. **Max iterations circuit breaker**: `while iteration < max_iterations` (default 10)
4. **Timeout + retry**: `mcp_client.call_tool` has `timeout=30s`; single retry on network error
5. No secrets in system prompt; prompts are static and deterministic

System prompt (static):
> "You are a vehicle safety research assistant with access to NHTSA data. Use the provided tools to answer questions. Always cite NHTSA as the data source. Do not speculate beyond what the tools return."

## Testing Strategy

### Unit (`tests/unit/`)
- **`test_inputs.py`**: VIN boundary tests, model_year boundaries, state allowlist, ZIP+4 normalization, lang canonicalization, miles bounds
- **`test_rate_limiter.py`**: 60 global calls pass, 61st → 429; 10 VIN calls pass, 11th → 429; daily quota; time-window expiry via mock clock
- **`test_cache.py`**: miss → fetch_fn called; hit → fetch_fn not called; TTL expiry; concurrent coroutines race on same key → fetch_fn called exactly once
- **`test_tools/`**: Each tool tested with `respx` mocking `httpx.AsyncClient`; success path, upstream 5xx → 502, upstream timeout → 504, cache hit (upstream called once)

### Integration (`tests/integration/`) — skipped by default
Live calls to real NHTSA endpoints; gated by `@pytest.mark.integration`.

### CLI (`tests/cli/`)
- **`test_cli_smoke.py`**: `typer.testing.CliRunner`; all subcommands register; invalid VIN → non-zero exit
- **`test_llm_stub.py`**: `StubLLMProvider` returns predictable tool call sequence; circuit breaker fires; unknown tool names blocked
